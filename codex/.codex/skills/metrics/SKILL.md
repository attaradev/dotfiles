# Metrics & Measurement Plan

Use this skill to turn a topic into a complete measurement plan.

## Workflow

1. Read the topic and the available metrics or analytics context.
2. Define a north star metric, input metrics, output metrics, guardrails, and anti-metrics.
3. State how each metric will be collected and reported.
4. Follow the framework in `references/metrics-framework.md`.

## Quality rules

- The north star must reflect user value.
- Input metrics must be actionable.
- Guardrails must cover engagement health, support load, and business safety.
- Every metric must have a measurement method.

## Resources

- `references/metrics-framework.md` provides the metric taxonomy and selection guidance.
