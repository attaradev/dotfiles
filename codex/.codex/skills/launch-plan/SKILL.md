# Go-To-Market Plan

Use this skill to plan a launch that reaches the right audience with the right message at the right time.

## Workflow

1. Read the launch topic and any existing launch or rollout docs.
2. Define the target audience, positioning, and launch phases.
3. Specify channels, success criteria, and risks for each phase.
4. Follow the structure in `references/gtm-template.md`.

## Quality rules

- Positioning must be understandable to a non-technical audience.
- Each phase needs entry and exit criteria.
- Channels must be specific enough to execute.
- Success criteria must be measurable before launch.
