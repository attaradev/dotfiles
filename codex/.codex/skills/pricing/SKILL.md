# SaaS Pricing Strategy

Use this skill to design pricing that matches customer value and the go-to-market motion.

## Workflow

1. Read the product context and current pricing situation.
2. Choose the pricing model and acquisition motion.
3. Define tiers, value metric, packaging, and price testing.
4. Follow `references/pricing-guide.md`.

## Quality rules

- The value metric must correlate with customer success.
- Every tier gate should map to a real buyer segment.
- The enterprise tier must have a clear reason to exist.
- Price testing should be explicit, not implied.
