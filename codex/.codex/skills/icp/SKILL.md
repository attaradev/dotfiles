# Ideal Customer Profile

Use this skill to define the customers who get the most value and buy for the right reasons.

## Workflow

1. Write a one-sentence ICP statement.
2. Specify firmographic, behavioral, and situational attributes.
3. Define the anti-ICP.
4. Create a simple scoring rubric to qualify or disqualify prospects.

## Quality rules

- Be specific enough to be useful.
- Make the situational trigger explicit.
- Include the anti-ICP.
- If you cannot name real customers that fit, the ICP is too abstract.
