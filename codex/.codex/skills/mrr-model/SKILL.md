# SaaS MRR / Financial Model

Use this skill to build a transparent SaaS financial model from the available data and assumptions.

## Workflow

1. Read the financial context and the question being asked.
2. State all assumptions explicitly.
3. Build the MRR waterfall, unit economics, retention analysis, and forecast scenarios.
4. Follow the structure in `references/mrr-model-guide.md`.

## Quality rules

- Show the waterfall, not only the net result.
- Make every assumption visible.
- Call out NRR and CAC payback benchmarks explicitly.
- Use different assumptions for scenario analysis.

## Resources

- `references/mrr-model-guide.md` provides formulas, benchmarks, and forecast structure.
